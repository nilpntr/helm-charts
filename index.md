# Helm Charts by nilpntr

Just another repo with some nice helm charts

## TL;DR
```bash
$ helm repo add nilpntr https://nilpntr.github.io/helm-charts/
$ helm search repo nilpntr
$ helm install my-release nilpntr/<chart>
```
